# SentinelForgeAI – Content Authenticity Engine

## 🚀 Overview
SentinelForgeAI Auth Detector analyzes text and predicts whether it is AI-generated or human-written.

## ⚙️ Tech Stack
- Python
- Flask
- Transformers (HuggingFace)
- PyTorch

## 🔍 Features
- AI vs Human text classification
- Confidence score output
- API-ready backend

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
